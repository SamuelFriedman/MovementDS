# Notation

Preliminary movement notation (inspired by EWMN, but with ASCII in mind)

< Leftwards
> Rightwards
^ Upwards
v Downwards

(x y)
[x y]

Angles
+ (Third)
++ (Half)
# (Two thirds)

Directions
-> is one path

^6 >3 is one movement amount

Rotations:
{v} Clockwise (CW)
|v| Counterclockwise (CCW)
